# Advent of Code 2025

Solutions for Advent of Code 2025, the "Short n' Sweet"-version.

## How to run

Replace `py` with whatever python-runner you have.

```shell
py runall.py   all # runs all
py runall.py     3 # runs only day 3
py runall.py  1-10 # run day 1 to 10
py runall.py 3,4,5 # runs days 3, 4 and 5
```
